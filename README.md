ArmyTool
ArmyTool is a modern, fast, and privacy-focused collection of online developer and utility tools.

Built for developers, students, sysadmins, creators, and everyday users who need quick access to useful utilities without unnecessary ads, trackers, or complexity.

✨ Features
JSON Formatter & Validator
JWT Decoder
Base64 Encoder / Decoder
UUID Generator
Hash Generator
URL Encoder / Decoder
Markdown Preview
Regex Tester
Unix Time Converter
QR Code Generator
Color Converter
And many more useful tools
🚀 Live Demo
https://www.armytool.site/

🛠️ Tech Stack
Vue 3
TypeScript
Vite
TailwindCSS
📦 Installation
git clone https://github.com/jobinblancaflor/IT-tools.git

cd armytool

npm install

npm run dev

🏗️ Build for Production
npm run build
🌐 Deployment

ArmyTool can easily be deployed on:

Vercel
Netlify
Cloudflare Pages
Docker
Self-hosted servers
🔒 Privacy

ArmyTool is designed to work mostly client-side whenever possible.

Your data stays in your browser and is not stored on our servers.
